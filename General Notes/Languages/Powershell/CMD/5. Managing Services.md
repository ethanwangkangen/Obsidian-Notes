# Service controller
- `sc`
	- Can query services, see permissions, etc.

## Query services
- `sc query type= service`
	- spacing important
- `sc query windefend`
	- query for Windows Defender

## Stopping and starting
- `sc stop windefend`
- `sc start Spooler`

## Modifying services
- Change them to be disabled at startup, modify service's path to binary, etc. etc.
- `config` parameter in sc
- eg. want to prevent updates
	- Windows update features relies on 2 services
		- `wuauserv` - Windows Update Service
		- `bits` - Background INtelligent Transfer Service
	- Stop it with
		- `sc stop bits`
	- Then disable future starts with
		- `sc config bits start= disabled`

# Other ways to query services

## Tasklist
- `tasklist /svc`

## Net start
- `net start`
	- no service specified -> lists all
	- service specified -> starts it
- Can stop, pause, continue with
	- `net stop`, `net pause`, `net continue

## WMIC
- Windows Management Instrumentation Command (WMIC)
- `wmic service list brief`
