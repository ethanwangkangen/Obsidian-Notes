## dir
- List directory contents.
- Examples:  `dir /a      (all incl. hidden/system)
           `dir /a:h    (hidden)   dir /a:r  (read-only)
           `dir /s      (recurse)  dir /o:n  (sort by name)
## cd
- Show/change current directory. Use /d to switch drive.
- Examples:  `cd          (print CWD)
           `cd /d D:\tools
           `cd ..        (up one)
## tree
- Show folder tree; /F includes files.
- Examples:  `tree
           `tree /F C:\Users\Public

## mkdir
- Create directory (alias: md).
- Examples:  `mkdir logs  |  md C:\temp\work
## rd /S
- Remove directory tree (alias: rmdir). Add /Q to suppress prompts.
- Examples:  `rd /S folder
           `rmdir /S /Q C:\temp\old
## move
- Move/rename files or folders.
- Examples:  `move file.txt D:\drop\
           `move oldname.txt newname.txt
           `move C:\dir1 C:\dir2\dir1
## xcopy
- Copy files/folders (legacy—prefer robocopy).
- Examples:  `xcopy src dst /E       (incl. empty dirs)
           `xcopy src dst /K       (keep attributes)
           `xcopy src dst /Y       (no overwrite prompt)
## robocopy
- Robust copy/sync. Syntax: `robocopy <src> <dst> [files] [options]
- Examples:  `robocopy C:\A C:\B
           `robocopy C:\A C:\B /E
           `robocopy C:\A C:\B *.log
- Options:
  /E        Recurse incl. empty dirs
  /MIR      Mirror (WARNING: deletes extras at dest)
  /COPY:DAT Copy data+attribs+timestamps (default)
  /DCOPY:DA Copy dir timestamps/attribs
  /R:1 /W:1 Retry once, wait 1s
  /A-:SH    Clear System+Hidden on dest
  /L        List what would happen (no changes)
  /B        Backup mode (needs privileges)
## more
- Page output one screen at a time; /S squashes blank lines.
- Examples:  `more big.txt
           `ipconfig /all | more
           `more /S long.txt
## type
- Print file(s) to console; doesn’t lock files.
- Examples:  `type readme.txt
           `type a.txt b.txt
- redirection with type
  Examples:  `type a.txt >> all.txt   (append)
           `type a.txt b.txt > all.txt  (overwrite)
           `type passwords.txt >> secrets.txt
## echo
- Print text; with > or >> to create/append files.
- Examples:  `echo hello > note.txt
           `echo more >> note.txt
           `echo. >> note.txt        (append blank line)

## fsutil
- Low-level NTFS tasks (admin). Useful here for quick files.
- file createNew
  Examples: `fsutil file createNew zero.bin 1048576  (1 MiB)
           `fsutil file createNew for-sure.txt 222

## ren
- Rename file or folder (alias: rename).
- Examples:  `ren old.txt new.txt
           `ren "Old Folder" NewFolder

## delete
- Deletes file/files
- Examples: `del <file>
	- /A:R - delete files with read-only attribute set
	- /A:H - delete files with hidden attribute set

## > | and >>
- Redirection & pipe.
- Examples:  `command > out.txt        (overwrite)
           `command >> out.txt       (append)
           `command < in.txt         (feed file to stdin)
           `producer | consumer      (pipe stdout to next cmd)
           `ipconfig /all | find /i "ipv4"
## &
- Run commands in sequence (don’t care if first fails).
- Examples: `ping 8.8.8.8 & type note.txt

## &&
- Run second command only if first succeeds.
- Examples: `cd C:\logs && dir
## ||
- Run second command only if first fails.
- Examples: dir C:\nope || echo "missing"
