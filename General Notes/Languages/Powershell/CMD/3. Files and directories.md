# Finding

## where (like which in linux)
- find file location
- eg. `where calc.exe
- \R - recursive
- *   - wildcard

## find (like grep in linux)
- find strings in files
- eg. `find "password" "C:\Users\student\not-passwords.txt" 
- /V: NOT clause. ie. find any line that does not have the string
- /N: show line numbers
- /I: ignore case sensitiviy

# Evaluating and sorting files

## compare
- compare differences in files
- eg. `comp .\file-1.md .\file-2.md

## fc.exe
- compares differences, but better
- eg. `fc passwords.txt modded.txt /N`
- for all options, use `fc.exe /?` to see 

## sort
- eg. `sort file1.md > sort1.md`
- /unique - to remove duplicates
