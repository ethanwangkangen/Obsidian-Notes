- `Get-Help *-Service`
	- To find cmdlets related to services

- `Get-Service`
	- List services
		- eg. `Get-Service | where DisplayName -like '*Defender*' | ft DisplayName,ServiceName,Status`
		- eg. `Get-Service Spooler`
	- `-ComputerName` to specify that we want to query **remote** host
		- eg. `Get-Service -ComputerName ACADEMY-ICL-DC | Where-Object {$_.Status -eq "Running"}`
- `Start-Service`
	- Start service
		- eg. `Start-Service WinDefend`
- `Stop-Service`
	- Stop service
		- eg. `Stop-Service Windefend`
- `Set-Service
	- Modify service
		- eg. `Set-Service -Name Spooler -StartType Disabled
			- Disable Startup

# Interacting with services on **remote** host
- `Get-Service -ComputerName ...`
	- See above
- `Invoke-command -ComputerName <name(s)> -ScriptBlock {<command>}
	- Run command(s) in `{..}`
	-  eg.`Invoke-command -ComputerName ACADEMY-ICL-DC,LOCALHOST -ScriptBlock {Get-Service -Name 'windefend'}
