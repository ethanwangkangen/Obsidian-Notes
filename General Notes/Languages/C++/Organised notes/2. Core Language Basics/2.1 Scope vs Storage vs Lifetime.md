# Storage (memory model)
- When is memory allocated and deallocated?
	- Automatic
		- Lives on stack
		- Created when function entered, destroyed when function exits
	- Static
		- Exists for entire program lifetime
		- Allocated before main, destroyed at program exit
	- Dynamic
		- Exists until `delete`
		- Heap managed
# Lifetime (runtime validity)
- When is object actually alive and usable?
	- Closely related to storage
- The `static` keyword gives a local variables static lifetime
	- However, it does not change it's scope.
