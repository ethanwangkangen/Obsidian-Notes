# Prefer alias
- Why? 
	- Templating
```cpp
template <typename T>
using MyAllocList = std::list<T, MyAlloc<T>>;

MyAllocList<Widget> lw;

// vs

template<typename T>
struct MyAllocList {
	typedef std::list<T, MyAlloc<T>> type;
};
```
- KIV. Item 9 of effective modern C++
- Alisas templates avoid the `"::type"` suffix, and in templates, avoid `"typename"` that typedefs require
- C++14 offers alias templates for all the C++ 11 type traits transformations
