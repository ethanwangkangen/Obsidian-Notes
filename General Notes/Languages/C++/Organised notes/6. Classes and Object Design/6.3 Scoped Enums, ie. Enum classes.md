# Scoped Enum (Enum classes)
```cpp
enum class Color{ //keyword 'class'
    red, // red is considered part of Color's scope region
    blue,
};

std::cout << Color::red;
```

- **No implicit conversion to int**, but can explicitly convert
```
std::cout << static_cast<int>(color) << '\n'; 
```
- More advanced way, overload the `operator+` to convert to int.
```CPP
template <typename T>
constexpr auto operator+(T a) noexcept
{
    return static_cast<std::underlying_type_t<T>>(a);
}
```

- Scope is contained within the class

- Can cast integer to scoped enumerator, useful when taking in input
```CPP
Pet pet{ static_cast<Pet>(input) }; // static_cast our integer to a Pet
```
