# CRTP
```cpp
template <typename Derived>
class Shape {
public:
    void draw() {
	    // Resoled at compile time
        static_cast<Derived*>(this)->drawImpl();
    }
};

class Circle : public Shape<Circle> {
public:
    void drawImpl() {
        std::cout << "Circle\n";
    }
};

class Square : public Shape<Square> {
public:
    void drawImpl() {
        std::cout << "Square\n";
    }
};

Circle c;  
c.draw();  
  
Square s;  
s.draw();

template <typename ShapeType>
void render(ShapeType& s) {
    s.draw();  // works for all CRTP types
}
```

- No virtual functions, no vtables, no runtime dispatch
- But keeps the benefit of **polymorphism**
- Can call generic `.draw()` interface function for all types that inherit from `Shape<>`
- Can call directly from object, no need base pointer/reference

- `static_cast<Derived*>(this)->drawImpl();`
	- Cast `this` pointer to type `Derived*`
	- Then can dereference to call the specific, implemented function `drawImpl()`
