# Passing by value
- Passing by value decays
	- eg. std::string, which has type `char const []`, decays to `char const*`

# Passing by reference
- Passing by reference does not decay
```cpp
template <typename T>
void foo(const T& arg){}

std::string const c = "hi";
foo(c); // T deduced as std::string, arg is const std::string&
foo("hi"); // T deduced as char[3], arg is const char (&) [3]
```
- Note for the above example: T is not const, the arg is
