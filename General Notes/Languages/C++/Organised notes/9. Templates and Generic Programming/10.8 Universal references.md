# Universal references
- Takes the form `T&&`
	- No `const` or other qualifiers
- However, not all `T&&` are universal references!
- eg.
```cpp
template <typename T>
struct Box {
    void f(T&& x);   // T is NOT deduced here — rvalue reference
};
```
- `T&& x` is **not** a universal reference, is a normal rvalue reference
	- Because it it not being **deduced** here.
		- `T` is fixed by the class tempalte, it was already deduced when `Box<T>` was instantiated.
