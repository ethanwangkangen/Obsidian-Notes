# std::vector vs std::array
- **`std::array<T, N>`**
    - Fixed-size array wrapper (size known at compile time)
    - Stack-allocated (usually)
    - No dynamic resizing
- **`std::vector<T>`**
    - Dynamic array
    - Heap-allocated storage
    - Can grow/shrink at runtime

## Performance considerations

- **`std::array`**
    - Faster access (no indirection, no heap)
    - No allocation cost
    - Best cache predictability
- **`std::vector`**
    - Slight overhead due to heap allocation
    - Amortized O(1) `push_back`
    - Possible reallocations → invalidates pointers/references/iterators

# Std::vector implementation
```cpp
template<typename T>
struct vector {
    T* first; // beginning of single memory allocation
    T* last; // pointer one past the end of last valid
    T* end; // pointer one past the end of the last allocated mem for vector
};
```
- size() is last-first
- capacity() is end-first

For interviews, 
```cpp
template<typename T>
struct vector {
    T* data;
	size_t size;
	size_t capacity;
};
```

# Full implementation
```cpp
template <typename T>
class vector {
public:
	vector() : vector(1) {
	}
	
	explicit vector(size_t capacity) : capacity_{capacity} {
		reserve(capacity);
	}
	
	// copy constructor
	vector(const vector& other) {
		copyFrom(other);
	}
	
	// move constructor
	vector(vector&& other) {
		stealFrom(std::move(other));
	}
	
	// copy and move assignment operator -> copy and swap idiom
	vector& operator=(vector other) {
		stealFrom(std::move(other));
		return *this;
	}
	
	void push_back(T elem) {
		expandIfNeed();
		new (data_ + size_) T(std::move(elem));
		size_++;
	}
	
	T& operator[](size_t index) {
		return data_[index];
	}
	
	const T& operator[](size_t index) const {
		return data_[index];
	}
	
	T& at(size_t index) {
		if (index >= size_) throw std::out_of_range("Out of range");
		return data_[index];
	}
	
	const T& at(size_t index) const {
		if (index >= size_) throw std::out_of_range("Out of range");
		return data_[index];
	}
	
	void pop_back() {
		size_--;
		data_[size_].~T();
	}
	
	bool empty() const {
		return size_ == 0;
	}
	
	size_t get_capacity() const {
		return capacity_;
	}
	
	size_t get_size() const {
		return size_;
	}
	
	// public!
	void reserve(size_t newCapacity) {
		std::allocator<T> myAllocator;
		
		// allocate memory
		T* newData = myAllocator.allocate(newCapacity);
		
		// copy/move over
		for (int i{}; i < size_; ++i) {
			new (newData + i) T(std::move_if_noexcept(data_[i]));
		}
		
		// delete old objects
		for (int i{}; i < size_; ++i) {
			try {
				data_[i].~T();
			} catch (...) {
				// do something
			}
		}
		
		// deallocate memory
		if (data_) myAllocator.deallocate(data_, capacity_);
		
		capacity_ = newCapacity;
		data_ = newData;
	}
	
	~vector() {
		while (!empty()) pop_back();
		std::allocator<T> myAllocator;
		myAllocator.deallocate(data_, capacity_);
	}

private:
	T* data_ {nullptr};
	size_t size_{};
	size_t capacity_{};
	
	bool needExpand() {
		return size_ >= capacity_-1;
	}
	
	void expandIfNeed() {
		if (needExpand()) {
			reserve(capacity_ * 2);
		}
	}
	
	void copyFrom(const vector& other) {
		std::allocator<T> myAllocator;
		// allocate memory
		T* newData = myAllocator.allocate(other.capacity_);
		
		// copy objects from other - DON'T MOVE!
		for (int i{}; i < other.size_; ++i) {
			new (newData + i) T(other[i]);
		}
		
		// delete old objects
		for (int i{}; i < size_; ++i) {
			data_[i].~T();	
		}
		
		// deallocate memory
		myAllocator.deallocate(data_, capacity_);
		
		data_ = newData;
		capacity_ = other.capacity_;
		size_ = other.size_;
	}
	
	void stealFrom(vector&& other) {
		std::swap(size_, other.size_);
		std::swap(capacity_, other.capacity_);
		std::swap(data_, other.data_);
	} 

};
```

## Reserve
- Must be **public** function
- **5 things** to do
	- Allocate memory
	- Move_if_no_except objects into new memory
	- Delete old objects
	- Dealloate old memory
	- Settle variables (capacity_ and data_)

## Copy and move constructors
- `copyFrom()` and `stealFrom()` respectively
- copyFrom
	- 5 things to do
	- Allocate memory (based on other's capacity_)
	- **Copy** objects over into new memory
	- Delete old objects
	- Deallocate old mmeory
	- Settle variables (capacity_, data_ and **size_**)
- stealFrom()
	- Just std::swap() every variable
