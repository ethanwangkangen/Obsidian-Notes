```cpp
#include <memory>

namespace getcracked
{
    // Implement here. Return std::unique_ptr<T>.
    template <typename T, typename... Args>
    std::unique_ptr<T> make_unique(Args&&... args) {
        return std::unique_ptr<T> (new T(std::forward<Args>(args)...));
    }
}
```
- Key concepts
	- Variadic arguments
	- `std::forward`
		- Such that for r-values, the r-valueness is preserved when forwarding to the constructor of T
	- Universal reference parameters
		- Such that for r-values, the move constructor is called instead of copy constructor (for the argument)
