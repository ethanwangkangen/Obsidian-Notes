# Member access
```cpp
std::tuple MyTuple{42, 3.14f, true};
std::cout << std::get<0>(MyTuple);
std::cout << std::get<bool>(MyTuple); // only works when there is ONE of that type, else compilation error
```

# Unpacking with structured binding
```cpp
std::tuple MyTuple{42, 3.14, "Hello"};
auto [a, b, c]{MyTuple};
```

## Tie() to update pre-existing variables
- std::ignore() to skip an element
```cpp
int a, b, c;
tuple AnotherTuple{100, 9.8, false};
std::tie(a, b, c) = AnotherTuple;

// to ignore the second element
std::tie(a, std::ignore, c) = AnotherTuple;
```

# Tuple type traits
*KIV: Type traits https://www.studyplan.dev/pro-cpp/type-traits*
- Size
	- `std::tuple_size_v<T>
- Element type
	- `std::is_same_v< std::tuple_element_t<...>, T> >`
```cpp
// Tuple size
template <typename T>
void HandleTuple(T Tuple){
  std::cout << "Tuple size: "
    << std::tuple_size_v<T>;
}


// Tuple elemnt type
template <typename T>
void HandleTuple(T Tuple){
  if constexpr (std::is_same_v<
    std::tuple_element_t<0, T>, int>) {
    std::cout <<
      "The first element is an integer";
  }
}
```

# Concatentaiton
- `std::tuple_cat()`
```cpp
std::tuple Original{42, 3.14, true};
std::tuple Additions{9.8f, "Hello"};

auto Combined{ std::tuple_cat(Original, Additions)};
```
