# Copy assignment 
- Say for `Obj` with member type `m_ptr` which points to `T` data.
```cpp
Obj operator=(const Obj& a) {
	// check self assignment
	if (&a == this) return *this;
	
	// delete resources we are holding
	delete m_ptr;
	
	// copy the resource
	m_ptr = new T; // create the object
	*m_ptr = *a.m_ptr; // copy construct the object
	
	return *this;
}
```

# Copy constructor
```cpp
Obj operator(const Obj& a) 
	: m_ptr{new T(*a.m_ptr)} {
}
```
