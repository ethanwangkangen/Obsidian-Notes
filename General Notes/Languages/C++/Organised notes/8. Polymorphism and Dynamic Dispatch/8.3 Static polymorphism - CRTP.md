# Curiously Recurring Template Programming
```cpp
template <typename Derived>
struct Elf {
    //virtual std::string_view Name() const = 0; 
    std::string_view Name() const {
        return static_cast<const Derived*>(this) -> Name();
    }
};

struct ArcherElf : Elf <ArcherElf> {
    //virtual std::string_view Name() const override { return "Legolas"; }; 
    std::string_view Name() const {
        return "Legolas";
    }
};
```
- Simplest example
- Key points
	- Base class **templated (use Derived)**
	- Within the function that you want polymorphism in,
		- Cast the `this` pointer to the **Derived** type
			- `static_cast<const Derived*>(this)` 
		- Then call the derived function
			- `-> function()`
