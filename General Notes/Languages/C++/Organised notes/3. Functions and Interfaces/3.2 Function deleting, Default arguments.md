# Default arguments
- `void foo(int y, int x = 5);`
- Use case: if may or may not have parameter to pass in.

```cpp
void printIDNumber(const int *id=nullptr) {
    if (id)
        std::cout << "Your ID number is " << *id << ".\n";
    else
        std::cout << "Your ID number is not known.\n";
}

printIDNumber(); // we don't know the user's ID yet

int userid { 34 };
printIDNumber(&userid); // we know the user's ID now
```

- Copy initialisation only

- Explicit arguments must be to the left

- Cannot be redeclared
	- Either in forward declaration and function declaration, **not both**
- Must be declared before use

- Not part of function's signature, so cannot be differentiated overloads
```cpp
void f(int x);
void f(int x = 10); // cannot overload like this. illegal redeclaration
```

```cpp
void f(int x = 10);
void f(); //this is ok, and preferred since exact match
```

- Don't work for functions through function pointers

> Best practice: **declare** default argument in **forward declaration** instead of functioon definition, since forward declaration is more likely to be seen by other files and included before use (espsecially if it's in header file)

## How it works, and tricky behaviour
- Default arguments are substituted by compiler at the **call site**,  not inside the function
```cpp
void f(int x = 10);
f(); // this is tranformed into f(10);
```

Trick 1: default arguments + **virtual** functions
```cpp
struct Base {
    virtual void f(int x = 1) {
        std::cout << "Base " << x;
    }
};

struct Derived : Base {
    void f(int x = 2) override {
        std::cout << "Derived " << x;
    }
};

Base* b = new Derived;
b->f();
```
- Default argument resolution happens at **compile time**
	- `b` is type `Base*`
	- So default argument comes from `Base` (static type)
	- The function call is converted to `f(1);`
- Virtual dispatch occurs at **runtime**
	- The actual function called is `Derived::f`
	- So `Derived::f(1);` is called

Trick 2: default arguments are **bound at declaration**, not definition
```cpp
void f(int x = 10);  // declaration

void f(int x) {      // definition
}
f(); // Calls f(10);
```

Trick 3: **function pointers** **ignore** default arguments
```cpp
void f(int x = 10);
auto p = f;

p(); // ERROR
// Because function pointer type is void(*)(int), not default arguments exist
```
