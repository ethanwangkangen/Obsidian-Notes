# Threads are move-only
```cpp
std::thread t1 (some_function);
std::thread t2 = std::move(t1);
```
- If a thread already has some associated thread, trying to move into it will terminate the program
	- For consistency with the `std::thread` destructor.
	- Must explicitly wait for a thread to complete/detach it before destruction
		- The same applies to assignment

## Can return threads from functions
```cpp
// rvalue returned
std::thread f() {
	void f();
	return std::thread(f);
}

std::thread g(f());
```
- Remember that thread only have move constructors
	- If we return from a function,
	- The return value is treated as an rvalue (standard says that return of a **local variable** treats expression as rvalue)
		- Hence move constructor works with it

# jthreads (C++ 20)
- Joining thread
- Automatically join in the destructor

```cpp
#include <thread>
#include <utility>
 
class joining_thread {
    std::thread t;
 
public:
    joining_thread() noexcept = default;
 
    template <typename Callable, typename... Args>
    explicit joining_thread(Callable&& func, Args&&... args)
        : t(std::forward<Callable>(func), std::forward<Args>(args)...) {}
 
    explicit joining_thread(std::thread t_) noexcept
        : t(std::move(t_)) {}
 
    joining_thread(joining_thread&& other) noexcept
        : t(std::move(other.t)) {}
 
    joining_thread& operator=(joining_thread&& other) noexcept {
        if (joinable())
            join();
        t = std::move(other.t);
        return *this;
    }
 
    joining_thread& operator=(std::thread other) noexcept {
        if (joinable())
            join();
        t = std::move(other);
        return *this;
    }
 
    ~joining_thread() noexcept {
        if (joinable())
            join();
    }
 
    void swap(joining_thread& other) noexcept {
        t.swap(other.t);
    }
 
    std::thread::id get_id() const noexcept {
        return t.get_id();
    }
 
    bool joinable() const noexcept {
        return t.joinable();
    }
 
    void join() {
        t.join();
    }
 
    void detach() {
        t.detach();
    }
 
    std::thread& as_thread() noexcept {
        return t;
    }
 
    const std::thread& as_thread() const noexcept {
        return t;
    }
};
```
