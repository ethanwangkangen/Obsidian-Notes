# 0/1 knapsack
- Choose a subset.. each element used **at most once**
- Capacity/target constraint
- Variants
	- Maximise value
	- Feasibility
	- Count ways

- State
	- `dp[j][w]` where w is the result, j is the index
	- We look at subset from `s[0:j]`

# Unbounded knapsack
- Use elements any number of times
- Coins, squares, rods
- `dp[w]` where w is the result. 
	- We don't care which elements we have already taken from the array, since we can take any number of times.
```cpp
// Combinations (Coin change II)
for (int coin : coins)  
	for (int i = coin; i <= amount; ++i)  
		dp[i] += dp[i - coin];
```

IMPORTANT
- Iterate through coins first before i!
	- Eg. coin change combinations, iterate through coins before i
	- To ensure no double counting
