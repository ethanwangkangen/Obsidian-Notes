## Classic (Find exact target)
```cpp
int l = 0, r = n - 1;

while (l <= r) {
    int mid = l + (r - l) / 2;
    if (nums[mid] == target)
        return mid;
    else if (nums[mid] < target)
        l = mid + 1;
    else
        r = mid - 1;
}

return -1;
```
- Identifier:
	- Have an **exact** value I want to find, want to see whether it's inside or not
## Left bound binary search (lower bound)
- FFFFF**T**TTTT 

```cpp
int l = 0, r = n - 1;

while (l < r) {
    int mid = l + (r - l) / 2;
    if (condition(mid)) // take left half
        r = mid; // mid can be the answer.
    else
        l = mid + 1; // mid cant be the answer.
}

return l;
```

- Identifier:
	- Want the **leftmost** value out of those that fulfil a certain condition

Eg. *Find minimum in rotated sorted array*
- Important: Think about what the "valid" condition is.
	- In this case, we want an element who is smaller than all elements to the right of it, ie. `nums[mid] < nums[right]`
		- Then take the leftmost out of all of this
## Right bound binary search (upper bound)
- TTTT**T**FFFFF
```cpp
int l = 0, r = n - 1;

while (l < r) {
    int mid = l + (r - l + 1) / 2; // REMEMBER THE +1 HERE!
    if (condition(mid)) // take right half
        l = mid; // mid can be the answer.
    else
        r = mid - 1; // mid cant be the answer.
}

return l;
```
- Identifier:
	- Want the **rightmost** value out of those that fulfil a certain condition

# Summary

| Type           | Goal         | Loop     | Mid                    | Update  |
| -------------- | ------------ | -------- | ---------------------- | ------- |
| Exact search   | find element | `l <= r` | `l + (r - l) / 2;`     | `±1`    |
| Left boundary  | first true   | `l < r`  | `l + (r - l) / 2;`     | `r=mid` |
| Right boundary | last true    | `l < r`  | `l + (r - l + 1) / 2;` | `l=mid` |
How to remember: **left** bound, condition valid, **right** changed. (vice versa)
Return **l** no mater what



Notes
- Initialise r to the **possibles values** of r, NOT just the array size!
	- eg. *Search Insert Position*. If appending to end of array, can be array.size(). So r is array.size() instead of array.size()-1
