# Vector
```CPP
# vector
vector<int> v;  
vector<int> v2(5); // size 5, default 0  
vector<int> v3(5, 10); // size 5, all 10
vector<int> v4{1,2,3,4,5};

v.push_back(x);  
v.pop_back();
  
v.size();  
v.empty();  
v.clear();  

v.reserve(n);
v.resize(n);

v.front();
v.back();

v.insert(v.begin() + i, x); // O(n)  
v.erase(v.begin() + i); // O(n)

v[i]; // O(1)  
v.at(i); // bounds-checked
sort(v.begin(), v.end());  
sort(v.rbegin(), v.rend()); //descending
```

# Hashset

# hash set
unordered_set<int> seen = {};
s.size();

# Hashmap

```cpp
# hash map
unordered_map<int, int> mp;

mp[key] = value;
mp[key]++; // auto-initialises to 0

mp[key] //inserts if not exists
mp.at(key) // throws if not exists

mp.count(key);
mp.contains(key);
mp.find(key); // returns iterator

mp.erase(key);
mp.erase(it);
  
mp.size();  
mp.empty();  
mp.clear();

for (auto &p : mp) {
    cout << p.first << " " << p.second;
}
```

# Queue
- Note: built upon `dequeue`
- No iteration
- No clear()
```cpp
queue<int> q;  
  
q.push(x); // enqueue  
q.pop(); // dequeue (no return)  
  
q.front(); // first element  
q.back(); // last element  
  
q.size();  
q.empty();
```
