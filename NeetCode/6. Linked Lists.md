## Lead lag pointers
- 2 pointers traverse linked list at same speed but different intervals
- Identifiers
	- Need to do things in 2 steps, or need to maintain access to an older node
```CPP
ListNode* lag = head;
ListNode* lead = curr->next->next; //(however many times we need this)
while (lead->next) { // lead will end at the last node.
	lead = lead->next;
	lag = lag->next;
}

```

## Fast Slow
- 2 Pointers traverse linked list at **different** speeds
- Identifiers
	- Getting middle of a linked list
	- Finding cycle
```cpp
ListNode* fast = head;
ListNode* slow = head;
while (fast and fast->next) { // Remember to check for fast->next!!!!!!
    fast = fast->next->next;
    slow = slow->next;
    // if (slow == fast) return false // if checking for cycle
}
return slow; // If returning middle of linked list

```
## Reversal
- Head becomes tail, all pointers reversed
```cpp
ListNode* curr = head;
ListNode* prev = nullptr;
while (curr) {
	ListNode* temp = curr->next;
	curr->next = prev;

	prev = curr;
	curr = temp;
}
return prev;
```

## Merging 2 linked lists
- Most important is how to create dummy node, then continually pick nodes from the existing lists to add to the result list.
```cpp
ListNode dummy{0};
ListNode* node = &dummy;

while (list1 and list2) {
	if (list1->val <= list2->val) { //take list1
		node->next = list1;
		list1 = list1->next;
    } else {
        node->next = list2;
        list2 = list2->next;
    }
    node = node->next;
}
if (list1) node->next = list1;
if (list2) node->next = list2;
return dummy.next;
```
